## デザインコンセプト

### テーマ
**「信頼 × 効率 × 安心」**

補助金・助成金という公的で重要な手続きを扱うシステムとして、ユーザーに「ここなら安心して任せられる」という信頼感を与えながら、操作の簡便さと処理スピードを体感させる。

### デザイン方針
- **Professional & Clean**: 行政・金融系に通じる清潔感のあるUI
- **Step-by-Step Guidance**: 複雑な申請プロセスを段階的に可視化
- **Feedback-Rich**: ボタン押下・ローディング・完了など全状態に明確なフィードバック
- **Mobile-First**: スマートフォンでも快適に操作可能

### スタイル
- Flat Design ベース + 控えめなシャドウ・グラデーション
- ダークモードは Phase 2 以降

## アクセシビリティ

### 基本方針
- WCAG 2.1 AA 準拠を目標
- キーボード操作でフル機能使用可能
- スクリーンリーダー対応（適切な aria-label 付与）

### コントラスト比
| 組み合わせ | 比率 | 合否 |
|---|---|---|
| `#1F2937` on `#FFFFFF` | 16:1 | ✅ |
| `#3B82F6` on `#FFFFFF` | 4.6:1 | ✅ |
| `#6B7280` on `#FFFFFF` | 5.9:1 | ✅ |

### フォーカスリング
```css
:focus-visible {
  outline: 2px solid #3B82F6;
  outline-offset: 2px;
  border-radius: 4px;
}
```

### フォームアクセシビリティ
- すべての `<input>` に `<label>` を関連付け（`for` / `id`）
- 必須フィールドに `aria-required="true"` + 視覚的な `*` マーク
- エラーメッセージに `aria-live="polite"` + `aria-describedby`

### モーダル
- 開いた時にフォーカストラップ
- `aria-modal="true"` / `role="dialog"`
- Escape キーで閉じる
- 閉じた後は元のフォーカス位置に戻る

## shadcn/ui コンポーネント利用方針

以下の shadcn/ui コンポーネントを優先利用：

| コンポーネント | 用途 |
|---|---|
| `Button` | すべてのボタン |
| `Input` | テキスト入力 |
| `Select` | プルダウン選択 |
| `Textarea` | 複数行テキスト |
| `Checkbox` | チェックボックス |
| `RadioGroup` | ラジオボタン（事業形態） |
| `Dialog` | モーダル全般 |
| `Badge` | ステータスバッジ |
| `Progress` | プログレスバー |
| `Card` | 補助金カード |
| `Separator` | 区切り線 |
| `Tooltip` | 補足説明 |
| `Toast` | 通知（完了・エラー） |

カスタマイズは `globals.css` の CSS 変数で一元管理。
